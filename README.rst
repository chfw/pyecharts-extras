Installation
================================================================================


You can install pyecharts-extras via pip:

.. code-block:: bash

    $ pip install pyecharts-extras


or clone it and install it:

.. code-block:: bash

    $ git clone https://github.com/pyecharts/pyecharts-extras.git
    $ cd pyecharts-extras
    $ python setup.py install
